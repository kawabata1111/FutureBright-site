  • ポータルサイト運用代行・管理
  • 返礼品開発・ブランディング・商品撮影
  • プロモーション戦略立案・実行
  • ECサイト掲載用デザイン制作
  • 大手ECモールでのテストマーケティング

03

人材育成・研修
プログラム開発

教育訓練機関認定
FutureBright株式会社は、厚生労働省認定の教育訓練機関として、一般企業職員向けの業務研修、教育訓練、カウンセリング及びコンサルティングを提供します。
企業や団体に合わせた社員研修プログラムを企画・提案・実施し、組織の成長を人の側面から支援します。

【研修名】生成AI活用講習

実務直結型。11時間の集中プログラムで、AIを使いこなす人材を育成します。

講習動画アジェンダ（全11時間）
01. イントロダクション (30分)
生成AIがもたらす業務変革 / データ種別ごとの活用可能性 / 講習全体の流れ
02. 生成AIの基本と操作 (60分)
ChatGPT・Gemini基本操作 / AI検索概要 / 基本ルールと注意点
03. 効果的なプロンプト設計 (30分)
実践的なプロンプトの書き方 / 実演例による改善ポイント解説
04. AIで学ぶ：習得と業務改善 (60分)
学習サイクルの作成 / 業務フロー改善ポイントの抽出
05. メール対応の効率化 (60分)
定型メール自動化 / 敬語・トーン調整 / 定型処理エージェント構築
06. 検索効率化・情報収集 (30分)
質問ベース検索 / Web解析と要点抽出 / 検索自動化テンプレ
07. 情報整理・要約の自動化 (30分)
長文要約・整理法 / 整理能力比較 / 情報整理エージェント構築
08. 会議準備・議事録化 (30分)
会議前資料生成 / 議事録自動化 / 議事録自動化AI構築
09. 文書・資料作成 (60分)
提案書ドラフト作成 / スライド・インフォグラフィック構成案 / 資料テンプレ作成AI
10. タスク管理・ＰＪ計画 (60分)
優先順位付け / スケジュール自動作成 / レポート自動化
11. ブレスト・アイデア創出 (30分)
アイデア自動提案 / 発想比較 / アイデア生成AI構築
12. 音声解析・生成 (60分)
文字起こし・要点整理 / 自然音声合成 / 多言語対応活用例
13. 動画解析・生成 (60分)
動画要約 / 動画生成AI(Veo 3.1等)の概要と実践 / ビジネス利用例
14. 生成AIと共に働く心得 (30分)
出力検証・フェイク対策 / 機密データ保護 / 役割分担
15. まとめと次のステップ (30分)
全体振り返り / AI活用ロードマップ / 明日からのアクション
受講後の成果イメージ

社内で生成AIを使える人財が増え、生産性・効率性が飛躍的に向上します。
